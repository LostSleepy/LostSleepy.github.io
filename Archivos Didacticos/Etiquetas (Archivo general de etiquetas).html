Etiquetas de Encabezado (Heading)

El elemento <h1> se usa para encabezados principales y es el más importante. Se utiliza para títulos.

Etiqueta de Encabezado 2

Los elementos <h2> hasta <h6> se utilizan para encabezados secundarios de menor importancia.

El elemento <p> se usa para párrafos de texto.

Enlace (Anchor)

La etiqueta <a> se utiliza para crear hipervínculos a otras páginas web o recursos.

La etiqueta <img> se usa para mostrar imágenes en la página web.

La etiqueta <ul> se utiliza para crear listas desordenadas, como listas de viñetas.

La etiqueta <table> se utiliza para crear tablas, mientras que <tr>, <th> y <td> se usan para definir filas, encabezados de tabla y celdas de datos, respectivamente.



La etiqueta <form> se utiliza para crear formularios web que permiten a los usuarios ingresar datos. Los elementos <label> describen los campos del formulario, mientras que <input> se usa para campos de entrada.
